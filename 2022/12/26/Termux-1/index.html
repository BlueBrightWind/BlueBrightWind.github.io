什么是 Termux

Termux 是一个 Android 终端仿真应用程序，用于在 Android 手机上搭建 Linux 环境， 不需要 root 权限 Termux 就可以正常运行。Termux 可以实现 Linux 下的许多基本操作，可以使用 Termux 安装 python，并实现 python 编程。最重要的是，可以在 Termux 中安装容器，实现完整的 Linux 系统。
